## คำอธิบายประกอบ 
- ### สัญลักษณ์คีย์บอร์ดบน macOS
    - Command หรือ Cmd (⌘)
    - Option (⌥)
    - Shift (⇧)
    - Control (⌃)

- ### ศัพท์เฉพาะ 
  - ↑ (ลูกศรชี้ขึ้น) = ปุ่ม Up Arrow  
  - ↓ (ลูกศรชี้ลง) = ปุ่ม Down Arrow 
  - PgUp = ปุ่ม PageUp
  - PgDown = ปุ่ม PageDown
